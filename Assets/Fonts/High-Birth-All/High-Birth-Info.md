# High Birth
High Birth is a 9pt pixel art font designed to have regal lettering giving it an elegant and luxurious feel. 

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 9px
- Scales: 9, 18, 27, 36, etc
- Ascender Max: 5px
- Descender Max: 4px
