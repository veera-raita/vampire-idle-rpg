# Royal Decree
Royal Decree is a 15pt pixel art font designed to resemble Gothic lettering.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 15px
- Scales: 15, 30, 45, 60, etc
- Ascender Max: 5px
- Descender Max: 3px

## Kerning
None
