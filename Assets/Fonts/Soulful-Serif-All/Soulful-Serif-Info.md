# Soulful Serif
Soulful Serif is a 12x10 pixel art font designed with heavily serifed lettering at a minimal scale.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 10px
- Scales: 10, 20, 30, 40, etc
- Ascender Max: 4px
- Descender Max: 4px
